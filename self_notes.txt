Github
  git config --list                                 - show config properties
  
  git config user.name "Your name"                  - set name value
  git config user.email "email@test.com"            - set email value

  git config --show-origin user.name                - active name
  git config --show-origin user.email               - active email


VS Code shortcut keys
  shift + alt + arrow keys                          - duplicate line of code
  alt + arrow keys                                  - move line of code
